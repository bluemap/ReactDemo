# 学习目标
1、熟悉React基础命令
2、新建简单demo
# 新建项目
npm install -g create-react-app     // 全局安装react
create-react-app demo1              // 创建demo1项目

如果npm 版本是5.2.0+以上，也可以用如下命令新建项目
npx create-react-app demo1

# 运行项目
npm start                           // 实际执行package.json中的react-scripts start命令

# 构建项目
npm run build
